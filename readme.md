# ⏱️ Таймер выключения / сна (PCTimer)

Десктопное приложение для отложенного выключения или перевода компьютера в спящий режим.  
Скачать ехе можно по ссылке [cкачать](https://github.com/K-A-R-P/PCTimer/releases/latest/download/PCTimer.exe) 

## ✨ Возможности

- ⏲️ **Гибкая настройка времени** – отдельные ползунки для часов и минут (от 1 минуты до 24 часов 59 минут)
- 🔘 **Переключатель режима** – выключение или спящий режим с визуальной подсветкой активной кнопки
- 🌐 **Два языка** – русский и украинский (переключаются одной кнопкой)
- 🔊 **Звуковое оповещение** – за минуту до срабатывания звучит сигнал (можно отключить)
- 🔔 **Предупреждение за 60 секунд** – окно автоматически разворачивается; действие можно отменить
- 📦 **Переносимость** – компилируется в один exe-файл (Windows) без лишних зависимостей

## 🧱 Требования

- Python 3.12
- Операционная система: Windows, Linux, macOS

Для уведомлений и звука используются только встроенные средства ОС (winsound / системный звонок), дополнительные библиотеки не требуются.

## ⚙️ Установка и запуск

### Клонирование репозитория
```bash
git clone https://github.com/K-A-R-P/PCTimer.git
```

### Установка зависимостей

Основная зависимость — **Flet**. Установите её через pip:
```bash
pip install flet
```
*Никаких других пакетов не нужно (plyer, winsound и пр. используются из стандартной библиотеки или необязательны).*

### Запуск из исходников
```bash
python main.py
```
При первом запуске Flet может загрузить дополнительные ресурсы – это однократная операция.

## 📦 Сборка в один исполняемый файл

Можно создать автономный .exe (или бинарник для вашей платформы) с помощью встроенной команды Flet:
```bash
flet pack main.py --name "PCTimer" --icon icon.png --add-data "icon.png;."
```
Если иконки нет, уберите `--icon icon.png`.  
Выходной файл появится в папке `dist`.

Для кастомной иконки подготовьте изображение `icon.png` (рекомендуется 256×256) в корне проекта.


## 🧩 Как это работает

1. Пользователь выбирает время (часы и минуты) и действие (выключение или сон).
2. Нажимает «Запустить» – начинается обратный отсчёт.  
   В главном поле отображается оставшееся время, обновляясь каждую минуту.
3. За 60 секунд до конца приложение издаёт звуковой сигнал (если включён) и окно разворачивается.
4. Если пользователь отменяет – интерфейс возвращается в исходное состояние.  
   Если нет – выполняется системная команда выключения или спящего режима (в зависимости от ОС).
